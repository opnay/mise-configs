# Mise Configuration Directory

PATH: `~/.config/mise/`

## `conf.d` Name rules

- 000 - 049: Mise core configurations
- 050 - 099: Setup and environment configurations
- 100 - 199: For development tools and languages
- 200 - 299: For Saas and web services (e.g. Github)
- 900 - 999: Temporary, experimental or only this device configurations (registered in .gitignore)
